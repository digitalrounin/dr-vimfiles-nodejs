dr-vimfiles-python
==================

Pedro's Node.JS vimfiles for use with homeshick.

## Requirements

  - [dr-vimfiles][dr_vimfiles]
  - [npm][npm]
  - [Node.js][node-js]
  - [Homeshick][homeshick]


### MacPorts installation

    $ sudo port install nodejs npm
    $ npm install instant-markdown-d
    $ lsdkfjsdf

[node-js]: https://nodejs.org/
[npm]: https://www.npmjs.com/
